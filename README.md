# nodetester

## Install
<pre>
git clone git@github.com:mikarinneoracle/nodetester.git
cd nodetester
npm install
</pre>

## Running
<pre>sh startNodemon.sh</pre>

## Testing
<pre>curl "http://localhost:3000"</pre>

## Unit Testing
<pre>npm test</pre>

## Swagger
<pre>http://localhost:3000/swagger</pre>
